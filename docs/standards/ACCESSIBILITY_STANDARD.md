# Accessibility standard (Kinetix web)

This file is the **in-repo** accessibility reference for the Kinetix product. It exists so markdown links under `docs/standards/` resolve inside this repository (standalone clone or nested monorepo).

## Baseline

- **Target:** WCAG 2.2 Level AA for primary flows in `apps/web`.
- **Audit log:** [`../ACCESSIBILITY_AUDIT_KINETIX_WEB.md`](../ACCESSIBILITY_AUDIT_KINETIX_WEB.md) (findings, route inventory, remediation notes).
