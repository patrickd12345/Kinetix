# @bookiji-inc/platform-auth (optional reference)

Shared admlog (one-shot admin dev login) for apps that choose to use it. **Not enforced** from the workspace root; no repo is required to depend on this package.

- **If a repo wants to use it:** depend on this package from that repo (e.g. via relative path or published package) and run install/build from that repo.
- **If unused:** this folder is kept as reference/archive. Do not delete without checking for any remaining references.

The workspace root does not list this package in a workspace; each repo is independent.
